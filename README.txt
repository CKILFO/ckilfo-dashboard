# Guide - Dashboard CKILFO avec MetaMask

## ✅ Fonctionnalités incluses
- Connexion MetaMask
- Vérification du bon réseau (BSC Testnet)
- Affichage de l'adresse du wallet connecté
- Affichage de la balance CKF (token ERC20)

## 🛠️ Instructions
1. Ouvre `index.html` dans un navigateur avec MetaMask installé.
2. Clique sur "🔗 Connecter MetaMask".
3. Autorise la connexion au wallet.
4. Assure-toi d'être sur **Binance Smart Chain Testnet (Chain ID: 97)**.
5. L'adresse du wallet et la balance en CKF s'affichent.

## 🌐 Déploiement GitHub Pages
- Dépose les fichiers dans un repo GitHub.
- Active GitHub Pages (branche `main`, dossier `/root`).
- Accède à ton dashboard en ligne 🎉
